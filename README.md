# ParticleSystem
gravity simulation

## keys & mousewheel
 - Esc
   Leave the program
 - four direction arrow / wsad
  Move viewport
 - +/c
   add more particles
 - -/x
   remove some particles
 - q
   slow down
 - e
   speed up
 - page up / mouse scroll upward
   zoom in
 - page down / mouse scroll downward
   zoom out
